5.How to deal with missing values?

While using the “TEDS 2016” dataset, missing values are a significant factor that needs to be taken care of while performing statistical analysis on the given data. If not handled carefully, missing values might have a considerable effect on the statistical analysis performed on the data.

While performing statistical analysis on a given dataset, missing values might be handled using several different techniques.

First, missing values might be completely removed using functions like “na.omit()” or “drop_na().”

Second, while performing statistical analysis like finding summary statistics for a given variable, missing values might be completely ignored by using “na.rm = TRUE.”

Third, while working with a survey-based dataset like “TEDS,” values like “98” and “99” might indicate “don’t know” or “refuse to answer,” so these values must be converted to “NA.”

Finally, missing values might also be handled using techniques like imputation, but for exploratory analysis, missing values might be completely removed or replaced with “NA.”
